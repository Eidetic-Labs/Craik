# Code of Conduct

Craik follows the Contributor Covenant Code of Conduct, version 2.1.

The full text is available at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Project Pledge

We are committed to making participation in Craik respectful, professional, and welcoming.

Expected behavior includes:

- using respectful and inclusive language,
- accepting constructive feedback,
- focusing criticism on ideas and work rather than people,
- respecting different experience levels and viewpoints,
- and helping keep project spaces productive.

Unacceptable behavior includes:

- harassment or intimidation,
- personal attacks,
- sustained disruption,
- sexualized language or imagery,
- publishing private information without permission,
- and other conduct that maintainers determine is inappropriate in project spaces.

## Enforcement

Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior. Maintainers may remove comments, reject contributions, temporarily restrict participation, or permanently ban contributors when necessary.

Reports should be sent privately to:

security@eideticlabs.ai

If that contact changes, this file should be updated before the project begins accepting broad external contribution.

This Code of Conduct is based on the Contributor Covenant, version 2.1.

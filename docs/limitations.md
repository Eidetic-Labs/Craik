# Limitations

Craik is pre-0.1.0.

Current limitations:

- The package and CLI scaffold exist, but runtime workflows are not implemented yet.
- No project registry, local state store, policy engine, memory backend, runner adapter, receipt store, or handoff writer is available yet.
- The CLI does not request tool authority or write runtime state.
- Stigmem integration is planned but not implemented in this package scaffold.

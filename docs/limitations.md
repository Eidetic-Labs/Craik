# Limitations

Craik is pre-0.1.0.

Current limitations:

- The package and CLI scaffold exist, but runtime workflows are not implemented yet.
- The local SQLite store exists for typed runtime records, but no project registry, policy engine, memory backend, runner adapter, receipt workflow, or handoff writer is available yet.
- The CLI does not request tool authority or write runtime state.
- Stigmem integration is planned but not implemented in this package scaffold.
